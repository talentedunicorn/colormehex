# colormehex

> Words are worth 16 million colors

# Usage

You could set values using a query parameter

- text value \
  `https://colormehex.talentedunicorn.com?text=Kevin%20Hart`
- birthday mode \
  `https://colormehex.talentedunicorn.com?text=Dad&birthday`
